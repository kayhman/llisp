A lisp like langage, with llvm backend
======================================

Features
--------
* Macro
* Closures
* Compiled (LLVM) and/or interpreted code
* Modular: core, string, kist, fucntional, compiler,  ... modules are loaded on request
* Three simple types : list, string, and float
